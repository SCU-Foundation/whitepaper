# SCU Whitepaper v1.8 — HTML Edition

This directory contains the full multi‑file HTML edition of the **SCU Whitepaper v1.8**, the current canonical architectural definition of the **Superconducting Compute Unit (SCU)**.

The HTML edition is structured for readability, deep linking, long‑term archival stability, and alignment with standards‑body publication practices.  
All files in this directory are part of the v1.8 publication artifact and are versioned, immutable, and checksummed.

## Structure

The HTML edition is organised into the following sections:

- **Front Matter** — mission, context, definitions, and guidance for reading the document  
- **Main Body** — core architectural content, generational roadmap, and systemic implications  
- **Appendices** — technical references, diagrams, scenarios, and supporting material  
- **Supplementary Information** — metadata, version history, institutional information  

Each section is stored in its own subdirectory for clarity and deep linking.

## Navigation

The HTML edition includes:

- a fixed left‑hand navigation sidebar  
- per‑page breadcrumbs  
- a “Return to SCU Foundation” link  
- consistent typography and ISO‑style layout  

All styling is provided by the edition‑specific stylesheet located at:
html/css/whitepaper.css


## Canonical Source

This HTML edition is the **canonical** version of the SCU Whitepaper v1.8.  
The SCU Foundation website links to this directory but does not host its own copies of the HTML files.
